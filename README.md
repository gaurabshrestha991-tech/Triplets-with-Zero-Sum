his program finds all unique triplets in an integer array whose sum is equal to **0**. It uses sorting and the two-pointer technique to efficiently avoid duplicate triplets.

## Concepts Used
- Classes and Objects
- Functions
- `vector`
- Sorting
- Two-Pointer Technique
- Loops
- Conditional Statements

## Features
- Finds all unique triplets that sum to zero.
- Avoids duplicate triplets.
- Efficient two-pointer approach after sorting.
- Time Complexity: **O(n²)**
- Space Complexity: **O(1)** (excluding the output array)
